### Note on changes
File `dolly.en.jsonl' contains examples from the original Dolly dataset with changed formatting.

Files `dolly.fr.jsonl', `dolly.pt.jsonl', and `dolly.ru.jsonl' contain examples from the Dolly dataset translated into French, Postugues, and Russian respectively, using NLLB-200-3.3B model (https://huggingface.co/facebook/nllb-200-3.3B).
